# 📈 Previsor de Demanda Semanal

Projeto desenvolvido em Python utilizando Streamlit para previsão de demanda semanal.

## Funcionalidades

- Média Móvel Simples
- Suavização Exponencial
- Regressão Linear
- Comparação entre métodos
- Gráfico comparativo
- Cálculo do MAE
- Recomendação gerencial

## Tecnologias

- Python
- Streamlit
- Pandas
- Plotly

## Como executar

```bash
pip install -r requirements.txt
streamlit run app.py
